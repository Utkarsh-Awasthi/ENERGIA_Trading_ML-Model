# ENERGIA_Trading_ML-Model
Built an end-to-end energy arbitrage ML pipeline using leakage-free DAM forecasts to predict IDA1 vs DAM direction in I-SEM half-hour markets. Applied time-based validation, profit-per-MWh evaluation, feature engineering, and visual analysis to produce a realistic trading strategy for Energia.
Achieved an overall accuracy of 81.5% in predicting profitable trading & overall profit of €139,403 in a day of trading by making effecient decisions.
